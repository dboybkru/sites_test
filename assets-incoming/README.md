# assets-incoming — приёмник картинок

Бросай сюда сгенерированные/готовые изображения. Дальше скажи ассистенту,
**для какого сайта/тренда** картинка — он оптимизирует её (webp/avif, нужные размеры),
перенесёт в `sites/<slug>/public/img/` и пропишет корректные пути под сборку `/s/<slug>/`.

Подсказки по именам (не обязательно, но помогает):
- `08-brutalism__hero.png` — префикс slug сайта + назначение
- или просто опиши словами, куда вставить

Форматы: webp/avif (фото — лучше всего), png (прозрачность), svg (вектор/лого), jpg (сконвертирую).

Эта папка — временная свалка. После переноса файлы отсюда можно удалять;
в гит коммитим уже оптимизированные версии в `public/img/` нужного сайта.
